# About This Project
This project is implementation of bidiretional streaming gRPC with simple chat usecase

# How to Run
- Clone this project
```
git clone https://github.com/marifsulaksono/go-chat-grpc.git
```
- Run server.go
```
go run app/server.go
```
- Run 1st client
```
go run app/client.go
```
- Run 2nd client
```
go run app/client.go
```
